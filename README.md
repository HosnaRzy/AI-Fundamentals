# AI-Fundamentals

Implementation and analysis of core artificial intelligence concepts including search algorithms, adversarial search, logical inference, and reinforcement learning

## Methodology & Technologies
- **Search Algorithms**: Uninformed (BFS, UCS, DLS) and informed (A*) from scratch with PyGame visualization
- **Adversarial Search**: Minimax with Alpha-Beta pruning for two-player competitive environments
- **Logical Inference**: First-order logic deduction using PyDatalog for constraint solving (Minesweeper)
- **Reinforcement Learning**: MDP with Value Iteration, Q-Learning, and Deep Q-Network (DQN) for stochastic environments
- **Evaluation**: Performance metrics including node expansion count, policy convergence, and inference accuracy

## Projects Overview
| Project | Description | Techniques |
| :--- | :--- | :--- |
| `Angry-Birds-Search` | Grid-based navigation to collect eggs with obstacles | BFS, UCS, DLS, A* |
| `Paint-Battle` | Competitive two-player territory capture game | Minimax, Alpha-Beta, Heuristics |
| `Minesweeper-FOL` | Logical agent solving minesweeper without guessing | PyDatalog, FOL, Inference Engine |
| `WALL-E-MDP-RL` | Stochastic environment with power management | Value Iteration, Q-Learning, DQN |
