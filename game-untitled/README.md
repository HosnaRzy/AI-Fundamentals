# Paint Battle - Adversarial Search Environment

## Project Overview
Competitive two-player grid game where agents paint territory to capture area. Implement adversarial search algorithms to create intelligent AI players.

### Core Environment
- **Grid-based**: Navigate and paint tiles on a grid with obstacles
- **Two Teams**: Compete to control more territory than opponent
- **Capture Mechanics**: Enclose areas to claim territory and eliminate opponents
- **Turn-based**: Alternating moves with real-time visualization

### Key Features
- Multiple agent types (human, basic AI, advanced AI)
- Customizable maps with different layouts
- Real-time Pygame visualization
- Territory scoring and player elimination
- Support for various search algorithms

### Implementation Tasks
1. **Adversarial Search**: Implement Minimax with depth limiting
2. **Optimization**: Add Alpha-Beta pruning for efficiency
3. **Heuristics**: Design evaluation functions for non-terminal states
4. **AI Agents**: Create intelligent opponents using search algorithms

### Learning Objectives
- Understand and implement adversarial search algorithms
- Design effective heuristic evaluation functions
- Optimize search with pruning techniques
- Handle complex game state representations
- Visualize and debug AI decision-making

**Technologies**: Python, PyGame, NumPy
